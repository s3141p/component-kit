# Ui devkit

# Prerequisites

1. Nodejs v14.15.1
2. `npm i -g @nrwl/cli` (Optional). Otherwise `nx` should be replaced with `npm run nx`

# Run app locally

1. `nx build builders`
2. `nx serve app -- --l=all`

# Running unit tests

`npm run nx test *libName*`

# Running lint

`npm run nx lint *libName*`
